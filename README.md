# 📝 todo-consoleapp

Простое консольное приложение для управления задачами, написанное на C#.  
Позволяет быстро добавлять задачи, отслеживать их статус и просматривать список прямо в терминале — без лишних зависимостей и интерфейсов.

## ✨ Возможности

- ➕ Добавление задач
- 🔄 Изменение статуса задачи
- 📋 Просмотр всех задач
- ⌨️ Навигация по меню с помощью клавиш

## 🚀 Установка

### Вариант 1 — Скачать готовый .exe (проще)

1. Перейди в раздел [Releases](https://github.com/fork1x/todo-consoleapp/releases)
2. Скачай `todo-consoleapp.7z` из последнего релиза
3. Распакуй архив в папку `todo-consoleapp`, где будет хранится приложение и задачи
4. Открой ярлык и пользуйся

> 💾 При добавлении первой задачи рядом с файлом `.exe` автоматически создастся файл `tasks.json` — в нём хранятся все твои задачи.

### Вариант 2 — Собрать из исходников (для разработчиков)

**Требования:** [.NET SDK 8.0+](https://dotnet.microsoft.com/download)

```bash
git clone https://github.com/fork1x/todo-consoleapp
cd todo-consoleapp
dotnet run
```

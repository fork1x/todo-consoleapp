# 📝 todo-consoleapp

> ⚠️ Проект находится в разработке. Функциональность будет расширяться.

Простое консольное приложение для управления задачами, написанное на C#.  
Позволяет быстро добавлять задачи, отслеживать их статус и просматривать список прямо в терминале — без лишних зависимостей и интерфейсов.

---

## ✨ Возможности

- ➕ Добавление задач
- 🔄 Изменение статуса задачи
- 📋 Просмотр всех задач
- 🎮 Навигация по меню с помощью клавиш
